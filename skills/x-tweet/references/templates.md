# Tweet Templates

## Single Tweet Templates

### The Build Update
> [What you built in one sentence].
>
> [One specific detail — time, tool, or result].
>
> Claude Code did most of the work.

### The Honest Fail
> [What went wrong].
>
> [What you learned from it].
>
> The fix took [time]. Worth it.

### The Tool Rec
> Been using [tool] for [time].
>
> [One specific thing it does well].
>
> Honestly can't go back.

### The Before/After
> Before: [old way, specific pain].
>
> Now: [new way, specific result].
>
> [One-line reflection].

### The Hot Take
> [Contrarian opinion in one sentence].
>
> [Why, in one sentence].

### The Metric
> This week: [2-3 specific numbers].
>
> All Claude Code.

### The Question
> [Genuine question about vibecoding/tools].
>
> [Why you're asking — one line of context].

## Self-Reply Templates

After main tweet, reply with one of:

- "For anyone asking how: Claude Code + [specific technique]. That's it."
- "The part I didn't mention: [the fail/struggle that preceded the win]."
- "If you want to try this: link in bio."
- "[Specific technical detail that power users will bookmark]."
- "Biggest surprise: [unexpected thing you learned]."

## Thread Template (3-5 tweets)

**Tweet 1**: Hook + main claim (must stand alone)
**Tweet 2**: The backstory / why this matters
**Tweet 3**: The specific how (tool, technique, approach)
**Tweet 4**: The surprise / what went wrong / unexpected lesson
**Tweet 5**: Question or call to action

## Content Calendar Alignment

Use the template that matches today's content type:
- Mon (Build Update): Build Update or Metric template
- Tue (Workflow Reveal): Thread template
- Wed (Hot Take): Hot Take template
- Thu (Tool Discovery): Tool Rec template
- Fri (Weekly Metrics): Metric template
- Sat (Deep Dive): Thread template
- Sun (Community): Question template + heavy replying
