# Hashtag Strategy

## Base Hashtags (rotate, max 2 per tweet)

| Hashtag | When to use |
|---|---|
| #VibeCoding | Default — use on most tweets |
| #ClaudeCode | When specifically about Claude Code features |
| #BuildInPublic | When sharing progress/metrics |
| #AI | Broad reach, use sparingly |
| #OpenSource | When promoting repos |

## Auto-Search Rules

Before posting, search X for:
1. Current trending hashtags in AI/coding space
2. Any new hashtags gaining traction in vibecoding community
3. Event-specific hashtags (conferences, launches, announcements)

Pick the most relevant trending tag to pair with one base hashtag.

## Rules

- Max 2 hashtags per tweet (algorithm reads content directly now)
- Place at end of tweet, not inline
- Never use hashtags that have < 100 recent posts (dead tags)
- Never use generic tags like #Tech #Coding #Developer alone
- If the tweet is a hot take or conversation starter, skip hashtags entirely (they reduce reply rate)
- Threads: hashtags on tweet 1 only, not every tweet
