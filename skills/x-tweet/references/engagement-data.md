# Engagement Data — X Algorithm (2026)

## Algorithm Weights

| Action | Multiplier vs Like |
|---|---|
| Reply | 13.5x |
| Retweet | 20x |
| Bookmark | 10x |
| Profile click | 12x |
| Link click | 11x |
| Conversation (reply + author reply back) | 150x |
| Negative action (block/mute/report) | -74x |

**Key insight**: Conversation depth dominates. One genuine reply chain > hundreds of likes.

## Posting Windows

| Window | Time (EST) | Why |
|---|---|---|
| Morning | 8-10am EST | Commute scrollers, highest initial velocity |
| Evening | 6-8pm EST | Post-work scrollers, second peak |

**First 3 hours**: 80% of total engagement happens here. Reply to EVERY comment in this window.

## Content Performance Rules

- Native video: ~10x more engagement than text-only
- Screenshots: ~5x more than text-only
- Under 110 chars: best for single tweets (easy scan, room for QT)
- Under 200 chars: good for standard tweets
- 280 chars: absolute max, only for pinned/important
- No links without X Premium: zero median engagement since Mar 2026
- 1-2 hashtags max: algorithm reads content directly now
- Reply-to-reply ratio: maintain 3:1 received/given

## Self-Reply Strategy

Always reply to your own tweet within 5 minutes:
- Adds context without cluttering the main tweet
- Starts conversation chain (150x multiplier)
- "Link in bio" goes here (until Premium)
- Follow-up hook: ask a question, add a detail, share the fail

## Thread Rules

- Max 5 tweets per thread (diminishing returns after)
- First tweet must stand alone (most people only see tweet 1)
- Each tweet adds ONE new idea
- Last tweet = call to action or question
- Number tweets: "1/5", "2/5" etc.
